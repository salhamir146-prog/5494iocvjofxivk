# پرتال بیمارستان امام علی

این مخزن شامل یک وب‌سایت استاتیک راست‌به‌چپ برای بیمارستان امام علی است. هدف پروژه ارائه یک نمونه پرتال بزرگ، مدرن و قابل توسعه برای معرفی خدمات بیمارستان، نوبت‌دهی، پزشکان، راهنمای بیمار و ارتباط با واحدهای درمانی است.

## فایل‌های اصلی

- `index.html`: صفحه اصلی پرتال بیمارستان.
- `assets/styles.css`: طراحی واکنش‌گرا و ظاهر گرافیکی سایت.
- `assets/app.js`: منطق تعاملی سمت کاربر.
- `pages/about.html`: معرفی بیمارستان.
- `pages/services.html`: معرفی خدمات تخصصی و دیجیتال.
- `pages/doctors.html`: فهرست نمونه پزشکان.
- `pages/patient-guide.html`: راهنمای مراجعه و بستری بیماران.

## وضعیت فایل‌ها در GitHub

اگر در صفحه اصلی مخزن فقط `.gitkeep` دیده می‌شود، مطمئن شوید آخرین commit این شاخه روی GitHub push شده است. فایل‌های سایت در همین شاخه track شده‌اند و شامل صفحه اصلی، assets، صفحات داخلی و workflow انتشار GitHub Pages هستند.

## انتشار روی GitHub Pages

فایل `.github/workflows/pages.yml` بعد از push کردن شاخه، سایت را اعتبارسنجی و به GitHub Pages منتقل می‌کند. فایل `.nojekyll` نیز باعث می‌شود GitHub Pages پوشه‌ها و assetها را بدون پردازش Jekyll منتشر کند.

## اجرای محلی

برای مشاهده سایت کافی است فایل `index.html` را در مرورگر باز کنید. برای اجرای اعتبارسنجی پایه:

```bash
npm test
```
